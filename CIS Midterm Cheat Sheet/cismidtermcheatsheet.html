Safiat's Cheat Sheet for Understanding HTML

Basic HTML Tags

<Doctype !>: This is used to specify the language and version of the document.

<html>: This defines that we are working with an HTML document

<head>: This gives information about the document.

<body>:This is the body of the document, where most of the txt is located.

<p>: This defines a paragraph.

Various Styling Tags

<b>: This is used to bolden text.

<strong> This is used to convey extra importance.

<big>: This is used to denote big text.

<i>: This is used to italicize text.

<small>: This is used to denote small text.

<sub>: This is used to denote subscripted text.

<sup>: This can be used to denote superscripted text.

<ins>: This is used to show inserted text.

<del>: This is used to show deleted text.

<h1> <h2> <h3> ... <h6>: These are all used to denote headings and their sizes are in descending order.

<a>: This represents a hyperlink, which can also be used to link to another document. It is shown by <a href="URL">

<abbr>: This is used to create abbreviations.

<br>: This is used to indicate line breaks.

<address>: This allows you to present the address in a web page.

<section> and <article >: You can nest article tags inside of section tags and vice versa.

<audio>: This attribute tells the browser to siplay controls so that users can play and control music.

<div>: This is a block section in a document, in which a line break is placed before and after.

<span>: Inline section in document.

Table Tags

<table>: This tag defines a table.

<tr>: This tag defines table rows.

<th>: This tag defines table header

<td>: This tag defines table row data

Form Tags

<form>: Forms are used to collect information from users.

<input>: This tag defines the type of input field (password, name)

HTML 5 Tags

<header>: This tag is used to place introductory content like logos, and author name. It belongs inside of the body tag.

<footer>: Footers usually contain information about the author, copyright data or links to other related documents.

<section>: This represents a standalone section.

<article>: This represents a self contained composition in a document which is intended to be independently reusable. It replaced the <div> element.

<nav>: This represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents. (eg. menu, table of contents, index)

<aside>: This represents a portion of a document which is only indirectly related to the document's main content. It is frequently presented as a sidebar or call-out box.

<audio>: This is the tag for embedding audio into a web page.

<video>: This element embeds a media player which allows video playback in the document. It can be used for audio content as well.

<progress>:This shows progress on the webpage as it loads.

Search Box Tags

<datalist>: This tag creates a list of pre- defined items.
